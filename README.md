# -Telegram
Бот для принятия отзывов от пользователей в свою группу

Всем привет! Я создал Telegram-Бота на языке Python версии 13.0+
# 1. Что он даёт?:
Данный БОТ предоставляет возможность принятия через него отзывов от пользователей их модерации и последующей отправки в группу

Команда /start:
Открывает меню, где можно оставить свой отзыв.

Команда /admin:
Админ-Панель с модерацией отзывов, управление группами и ограничения

Для изменения названия на 1 блоке с приветствием отредактируйте значение "Ваше название" на строках 102 и 116.

# 2. Как добавить Бота в группу?:
Добавляете Бота в группу через настроки участников в вашей группе, затем назначаете Бота администратором выдав ему все разрешения (Можно анонимность, добавив подпись типо:"БОТ")

# ВАЖНО: Зайдите в настроки Бота в @botfather, жмите на кнопку "Bot Settings" далее "Group Privacy" и убедитесь что там стоит значение "disabled".
